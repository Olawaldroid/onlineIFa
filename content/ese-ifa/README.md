# Ẹsẹ Ifá source shelf

This directory is the tracked provenance layer for verse material. The source
PDFs themselves live under `docs/originals/open-sources/` and remain ignored by
Git; `source-catalog.json` records their canonical URLs, rights posture, local
filenames, byte sizes, page counts, and SHA-256 hashes.

Only a passage in a dedicated, reviewed corpus file may reach the public site.
An archived PDF is **not** publication approval. Each public passage needs:

- a stable ID and named Odù;
- the performer/reciter and recording context when known;
- printed-page and PDF-page locators;
- exact rights and attribution;
- explicit excerpt/translation/orthography notes; and
- source, language, and practitioner review.

`pogoson-akande-2011.json` is the first public source record. Its quoted fields
retain the source article's CC BY-NC-SA 4.0 terms. The surrounding metadata and
editorial summaries are original project material.

The Lijadu scan is a high-value public-domain lead, but its old orthography,
missionary framing, and lack of a modern rights-clean translation mean it stays
in transcription review. We will not turn OCR into “authentic” Ẹsẹ automatically.
